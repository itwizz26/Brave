== Installation ==

Upload into the plugins directory and head over to the plugins section in the Admin panel,
activate "Earthquake Catalog API" and then you are done!

== What's next ==

Once the plugin is activated, go to the pages section.
There will be a page called "Earthquake Catalog" that was automatically created
when the plugin was activated.

Open the page and see earthquake data mapping across the world.
